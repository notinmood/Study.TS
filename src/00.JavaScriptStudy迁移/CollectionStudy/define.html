数组的3种构建方式
1 	var LangShen = [ "Name","LangShen","AGE","28" ];

当然也可以用Array的构造方法定义
1 	var LangShen = Array( "Name","LangShen","AGE","28" );
2 	var LangShen = new Array( "Name","LangShen","AGE","28" );

数组调用的时候，用[index]:
LangShen[0]    结果为Name

对象的3种定义方式（效果是一样的。其中第一种方法叫字面量定义）
let myObject = {'name': 'zhangsan', 'sex': 'male', 'age': 20};
let myObject = Object({'name': 'zhangsan', 'sex': 'male', 'age': 20});
let myObject= new Object({'name': 'zhangsan', 'sex': 'male', 'age': 20});

对象中属性的调用方式有两个方式
console.log(myObject.name);
console.log(myObject['age']);
